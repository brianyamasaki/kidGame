Description of the Code

  • This game of Tic Tac Toe is created in the ticTacToe directory
  • The game logic is in ticTacToe-directive.js and the game HTML template is in ticTacToe-directive.html.
  • The value of each square of the board is represented by array items within scope.boardValues. While JavaScript array items will accept all kinds of values, the only values that actually work are 'x', 'y', and the special value undefined. Note that the board is initialized to be full of undefined values by copying from initSquares.
  • Note: The little number displayed on each tic tac toe square of the game itself is only there to help the coder to remember which item in the array represents which square on the board.
  • scope.currentPlayer holds the value of the current player, always either 'x' or 'o'.
  • When the user clicks on a square, the function scope.squareClick gets called with an index parameter from 0 to 8. To mark that square for the current player do scope.boardSquares[index] = scope.currentPlayer. As an example, clicking on the top left square will call scope.squareClick with index equal to 0. If the current player is 'x', the correct way to display the board properly is to set scope.boardSquares[0] = 'x'.

Exercises

  1. Change the scope.resetBoard function to start with player 'o' instead of player 'x'.
  2. Disallow clicking on a square that is already marked by changing code in scope.squareClick. You'll want to check immediately whether it's already marked. Give an error message by setting scope.errorMsg = 'Cannot Click on this square', then return false before any other code is executed. If clicking on the square is allowed, be sure to set scope.errorMsg = undefined before continuing to execute the code.
  3. Finish the function scope.checkForTie by detecting for any squares that are undefined. Note that if you find any undefined array items, you can return false immediately. Otherwise, if you can't find any undefined array items, you should return true.
  4. Finish the function scope.checkForWin by checking whether any three in a row matches for the current player. There are eight possible ways to win - three across, three down and two diagonal. If any three in a row matches the current player, the function should return true. Otherwise return false. The simplest way to do check is to compare the array items with scope.currentPlayer such that three in a row are actually the same.
